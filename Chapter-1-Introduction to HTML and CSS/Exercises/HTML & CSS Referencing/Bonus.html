HTML STRUCTURAL TAGS

<section>

A structural HTML element used to group together related elements.


                <section>
                 <h2>Bathspa</h2>
                 <p>BATHSPA ACADEMIC CENTRE RAK</p>
                </section>

<div>

It is primarily used to group similar contents together so that styling it would be easy.


                <div class="Highlights">
                 <p>The Highlight of the website is...</p>
                </div>

<Header>

Usually Used for the website's Navigation or Introductory Content.

<Footer>

Usually contains copyright info, Links or contacts.

HTML Text Formatting

<p>

Paragraph Tag.


                <p>Web Development Class</p>

<h1>-<h6>

These are the heading tags, where <h1> is the most important tag and <h6> being the least important.


                <h1> The Main Heading </h1>
                <h6> The Smallest Sub-Heading </h6>

<strong>

Makes the text bold


                <strong>Important</strong>

<em>

Makes the text italic


                <em>Amazing</em>

CSS Tags

color

It's used to style the html and give the color for the text.


                .container{
                    color: red;
                }

padding

Used For Creating A Space Within An Element.


                .container{

                    padding:30px;
                }

Background-color

It's Used to change the background color of the page or element.


                body{
                    Background-color : aqua;
                }

border-radius

it's used to change the radius of a box or element


                .corner{
                    border-radius: 10px;
                }

font-family

It's used to change the font of the page, element or division.


                .fontstyle{
                    font-family: lobster;
                }
