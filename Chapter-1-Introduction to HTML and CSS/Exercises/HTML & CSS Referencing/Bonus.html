HTML STRUCTURAL TAGS

<section>

A structural HTML element used to group together related elements.


                <section>
                 <h2>Bathspa</h2>
                 <p>BATHSPA ACADEMIC CENTRE RAK</p>
                </section>

<div>

It is primarily used to group similar contents together so that styling it would be easy.


                <div class="Highlights">
                 <p>The Highlight of the website is...</p>
                </div>

<main>

It contains the main content. means the body, each html file only contains one main tag.


                <main>
                    <div>
                        <h2> hello </h2>
                    </div>    
                </main>

<Header>

Usually Used for the website's Navigation or Introductory Content.


                <header>
                    <h1> Welcome to my website </h1>
                </header>

<nav>

The navigation tag is used for navigation links. It can be nested within the <header> tag, but secondary navigation <nav> tags are also commonly used elsewhere on the page.


                <nav>
                    <a href=""></a>
                    <a href=""></a>
                    <a href=""></a>
                </nav>

<Footer>

Usually contains copyright info, Links or contacts.


                <footer>All rights Received © </footer>

HTML Text Formatting

<p>

Paragraph Tag.


                <p>Web Development Class</p>

<h1>-<h6>

These are the heading tags, where <h1> is the most important tag and <h6> being the least important.


                <h1> The Main Heading </h1>
                <h6> The Smallest Sub-Heading </h6>

<strong>

Makes the text bold


                <strong>Important</strong>

<em>

Makes the text italic


                <em>Amazing</em>

CSS Tags

color

It's used to style the html and give the color for the text.


                .container{
                    color: red;
                }

text-shadow

It's used to add shadow to a specific text


                .shadow{
                    text-shadow: 2px 2px 5px red;
                }

border-image

This is used to add image as a border-image


                .b-image{
                    border: 10px solid transparent;
                    padding:15px;
                    border-image:url(border.png) 20 round;
                }

padding

Used For Creating A Space Within An Element.


                .container{
   
                    padding:30px;
                }

Background-color

It's Used to change the background color of the page or element.


                body{
                    Background-color : aqua;
                }

border-radius

it's used to change the radius of a box or element


                .corner{
                    border-radius: 10px;
                }

font-family

It's used to change the font of the page, element or division.


                .fontstyle{
                    font-family: lobster;
                }

backround-image

This is used to add background image 


                .bg-img{
                    background-image: linear-gradient(to bottom right, red, yellow);
                }
